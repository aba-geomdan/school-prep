# 학교 준비반 자동화 시스템

검단ABA언어행동연구소 · ABA 기반 학교 준비반 보고서 자동 생성 시스템

회기별 평가만 입력하면 **5가지 보고서**(회기별 평가서·중간보고서·종결보고서·학교 전이 평가·보호자 안내문)가 자동으로 작성됩니다.

---

## 🚀 배포 (GitHub Pages)

### 처음 한 번만 (저장소 생성 직후)

```bash
# 1) 의존성 설치
npm install

# 2) GitHub Pages 활성화
# 저장소 Settings → Pages → Source: "Deploy from a branch" → Branch: gh-pages → /(root)
```

### 코드 수정 후 매번

```bash
# 빌드 + GitHub Pages에 자동 배포
npm run deploy
```

배포 URL: **https://aba-geomdan.github.io/school-prep/**

> ⚠️ 저장소는 비공개(private)이지만 GitHub Pages 사이트는 URL을 아는 사람이 접근 가능합니다. 링크 공유를 신중히 해주세요.

---

## 💻 로컬 개발

```bash
# 개발 서버 (http://localhost:5173)
npm run dev

# 빌드 결과 미리보기
npm run build
npm run preview
```

## 🔑 계정 정보

- **관리자**: 코드 내부에 하드코딩 (App.jsx 33줄)
- **치료사 계정**: 관리자 페이지에서 발급

---

## 📦 데이터 저장

모든 데이터는 사용 중인 **브라우저의 localStorage**에 저장됩니다. 서버 백엔드 없음.

### 백업 권장 주기
- **주 1회 이상**: 시스템 우상단 「📦 백업 내보내기」 클릭 → JSON 파일 다운로드
- 다른 컴퓨터·브라우저에서 작업하려면 「📥 백업 불러오기」로 복원

> 브라우저 캐시·쿠키 전체 삭제 시 데이터가 사라집니다. 백업은 필수입니다.

## 🛠 기술 스택

- **React 18** (단일 파일 App.jsx, ~11,000줄)
- **Vite** (번들러)
- **Recharts** (레이더 차트, 라인 차트)
- **GitHub Pages** (정적 호스팅)

별도 백엔드·데이터베이스·서드파티 API 없음.

---

## 📋 주요 기능

| 보고서 | 자동 생성 내용 |
|---|---|
| 📋 회기별 평가서 | 8영역 5점 척도 → 회기별 「잘한 점·개선할 점」 자동 작성 |
| 📊 중간보고서 | 월별 평균 변화 분석 → 기간 총평 + 다음 8주 IEP 목표 |
| 🎓 종결보고서 | 첫 vs 마지막 비교 → 영역별 변화 요약 + 다음 단계 권장 목표 |
| 🎒 학교 전이 평가 | 종결 점수 → 학교 적응 시나리오 + 입학 후 16주 IEP 목표 |
| 💌 보호자 안내문 | 회기 활동에서 가정 연계 활동 자동 제안 |

## ⚠️ 사용 안내

- 자동 생성된 모든 내용은 **그대로 사용하거나 직접 수정** 가능합니다
- 평가의 책임은 작성한 치료사에게 있으므로, **출력 전 반드시 검토**해주세요
